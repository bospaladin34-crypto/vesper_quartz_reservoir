# Braid Syntax Overview

Braid Syntax is a deterministic, block‑structured DSL.

A typical module:

[OPERATOR: SANTOSLAGRANGIANBENTHIC_V1]
PHASE_DELTA::= 0.17259029
PARITY::= MAJORANA-1

L0::= LAMINAR.KINETIC(dq/dt) - TURBULENT.POTENTIAL(q)
DIVTERM::= ∇·Fthermo

Core features:

- Block headers
- Key/value invariants
- Operator expressions
- Deterministic structure
