# Braid SDK

The **Braid Syntax Development Kit** is the official toolchain for working with the
VESPER Braid language — a deterministic, laminar, operator‑driven DSL used for
manifold configuration, tensor ingestion, and Santos‑Protocol physics modeling.

This documentation covers:

- The Braid language
- The Python SDK
- The Santos Lagrangian compiler
- The IntelliSense engine
- The VSCode extension
- Example modules
